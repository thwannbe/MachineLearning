Detailed Description

NAtive Mechanic Intelligent Machine Learning Framework Source File.

namiML framework supports multiple machine learning algorithms.
Some algorithms require different forms of inputs or training
data. Be careful not to adapt wrong type of data before executing
the program. The list of machine learning algorithms is in a header
file.
