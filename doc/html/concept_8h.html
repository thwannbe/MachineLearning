Detailed Description

Concept Learning Framework Header File.

Based on Candidate Elimination Algorithm, it predict target
concept. It assumes that input is a conjunctive set of attributes
including target attribute. All attribute has binary output ( T/F )

enum Result

Possible Hypothesis Result.

Candidate Elimination Algorithm predict new instance searching all
hypothesis space and voting their result for new instance. If they
output same result as true or false, it will result T / F. Otherwise
their result is not unanimous, then it will result dontknow.

Enumerator
r_false 

means all hypothesis vote false

r_true 

means all hypothesis vote true

r_dontknow 

means hypothesis vote is not unanimous
