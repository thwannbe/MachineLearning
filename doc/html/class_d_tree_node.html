Detailed Description

Decision Tree Node Class.

Decision Tree Learning has no exact form of hypothesis, but
the path which follows any tree node is one of hypothesis.
Tree node has a list for possible values which this attribute
can have, and a value which is target value means when this node
is a leaf. When this node is not leaf, value means next test attribute
index.
