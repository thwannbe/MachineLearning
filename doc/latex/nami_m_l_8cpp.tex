N\-Ative Mechanic Intelligent Machine Learning Framework Source File.  




\subsection{Detailed Description}
N\-Ative Mechanic Intelligent Machine Learning Framework Source File. nami\-M\-L framework supports multiple machine learning algorithms. \par
 Some algorithms require different forms of inputs or training \par
 data. Be careful not to adapt wrong type of data before executing \par
 the program. The following is the list of machine learning algorithms \par
 which are supported by this program and the input specifications.\hypertarget{nami_m_l_8h_list_section}{}\subsection{Lists}\label{nami_m_l_8h_list_section}
{\bfseries C\-O\-N\-C\-E\-P\-T} {\bfseries L\-E\-A\-R\-N\-I\-N\-G} -\/ B\-A\-S\-E\-D O\-N C\-A\-N\-D\-I\-D\-A\-T\-E E\-L\-I\-M\-I\-N\-A\-T\-I\-O\-N ; \par
 This assume that all input is a conjunction of binary \par
 attribute with a target attribute as its training data. \par
 Also training data must have no error. \par
 New instance which will be predicted should have same number \par
 of attributes like training data except for target attribute. \par
 All attributes must be clear(i.\-e. 1-\/true or 0-\/false). \par
