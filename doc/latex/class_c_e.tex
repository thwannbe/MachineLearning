\subsection{Detailed Description}
Candidate Elmination Algorithm Engine. 

\hyperlink{class_c_e}{C\-E} algorithm as a kind of concept learning algorithm. \par
 Defines two bound to represent version space, and predict \par
 comparing only bound hypotheses. Training examples are used for \par
 building two bounds.
